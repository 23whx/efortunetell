# 八字命理入门指南

八字命理是中国传统命理学的重要组成部分，通过分析一个人出生时的年、月、日、时四柱八字，来推测其一生命运走势。

## 基本概念

八字由天干地支组成，共八个字，故称"八字"。天干有十个：甲、乙、丙、丁、戊、己、庚、辛、壬、癸；地支有十二个：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。

## 入门知识

1. **排八字**：根据出生年月日时排出四柱八字
2. **十神分析**：比肩、劫财、食神、伤官、偏财、正财、七杀、正官、偏印、正印
3. **五行生克**：金、木、水、火、土之间的相生相克关系
4. **大运流年**：分析人生不同阶段的运势变化

## 学习建议

初学者应从基础理论开始，逐步掌握排八字、分析十神和五行关系等基本技能，再结合实例进行实践。
